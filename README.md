## 底部设置

### 第一步

首先登录网站后台 - 内容设置 - 页面管理，添加以下6个页面，内容自己写自己网站相关内容即可：

友情链接 links  
广告投放 advertise  
联系我们 contacts  
招牌信息 job  
版权声明 copyright  
About about

**左边页面标题，右边的英文是页面URL！！！**

### 第二步

将下面的6条规则加入到全局设置 - 站点设置 - URL 链接显示样式规则当中：

/page/about===/about/  
/page/advertise===/advertise/  
/page/contacts===/contacts/  
/page/job===/job/  
/page/copyright===/copyright/  
/page/links===/links/

**请保持网站伪静态开启！！！**
